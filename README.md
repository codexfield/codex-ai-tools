## 🚀 快速开始

### 在线体验

访问我们的 [showcase 网站](https://codex-ai-tools.vercel.app) 即可体验所有工具。

### 本地开发

```bash
# 克隆仓库
git clone https://github.com/codexfield/codex-ai-tools.git
cd codex-ai-tools

# 安装依赖
npm install

# 启动开发服务器
npm start
```
